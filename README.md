# 营销精准支撑

## 一、搭建精准营销受理流程营销前台
 * 营销方案报备，获得营销许可
 * 营销项目分类（新增、副卡、宽带、异网高价值）
 * 客户信息登记
 * 系统检验信息有效性
 * 信息上报到客户经理
 * 客户经理审核确认
 * 客户订单反馈
 * 信息导入VGOP系统
 * 动态反馈渠道销售情况
 

  ## 构建营销渠道适配规则

   * 短信通道适配
     - 目标人群：信息被动型用户，对营销信息不反感
